总结：

常用标签

内联元素：span / strong / em / lable / del(删除线) / ins(下划线) / a / sup(上标) / sub(下标)

块级元素：div / h1-6 / p / address / ul / ol / li / dl / dt / dd / table / form / fieldset / legend

内联块元素：input / img / select / textarea / iframe(内联框架)

HTML4.0 有89个标签(十分之3常用)，HTML5标签新增30个标签，二分之一是有用的，其他要不然是兼容性不行或者是功能性太强。不到一定web发展阶段是很难有出路的。
