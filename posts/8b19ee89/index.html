区别:

  1. 使用 return 出去后需要把执行函数赋值给一个全局变量,然后再执行这个全局变量的函数
  2. 使用 window 直接将函数保存到GO。既把函数保存到 window 上的一个变量,然后再执行 window 上变量所对应的函数。
