  • js中的 ||运算 和 &&运算

    2020-07-19
    js中的 || 运算

    只要 || 前面为 false 的话那么就会往后走(直到碰到true)，走到最后不管是真是假都返回。

    js中的 && 运算

    只要 && 前面为 false 的话,那么就返回 && 前面的值，不管你后面有多少个true。

    补充

    在javasript中 undefined,null,NaN,“ “,0,false 这些都是为false即假。

    举个栗子

    写一个函数如果有传入姓名则 return 出去，否则 return 您没有输入姓名
